VowKy 1.3.3 Release Notes

- Fixed auto-update failures on macOS Sequoia / Tahoe (the "Update Error! An error occurred while running the updater" message); updates now download and install correctly.
- Redesigned the "Update Available" reminder window: it now shows the current version, the latest version, and what's new more clearly.
- Speech recognition now runs in a separate process for greater stability: even if the recognition engine crashes, the main app is unaffected.
