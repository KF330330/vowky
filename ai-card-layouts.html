VowKy · AI 助手卡片布局方案对比

为「设置 → AI 助手」分区下的平台安装卡片（Claude Code CLI / Codex CLI）设计 5 种紧凑布局，每个方案覆盖 4 种状态：未装 / 已装最新 / 已装旧版可更新 / 冲突。

首选：方案 E（左侧状态色条 + 紧凑内容） — 色条提供周边视觉的整卡健康度，扫读不需要聚焦到文字；密度足够紧凑且错误展示位置自然（色条直接变红）。
